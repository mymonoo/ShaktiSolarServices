# Shakti Solar Services

A professional website for Shakti Solar Services, showcasing solar pumps, controllers, and services.

## Features
- Modern, responsive design.
- Detailed information on Solar Pumps, Controllers, and RMS (Remote Monitoring System).
- Support and query forms.

## Usage
Simply open `index.html` in your browser or host it on a static hosting service like GitHub Pages.

## Deployment
This project is ready to be hosted on GitHub Pages.
1. Push this code to a GitHub repository.
2. Go to Settings > Pages.
3. Select the `main` branch and `/root` folder.
4. Your site will be live at `https://<username>.github.io/ShaktiSolarServices/`.
